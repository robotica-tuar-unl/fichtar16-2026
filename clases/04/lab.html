2 Programación de nodos

  • Importar la librería rclpy

import rclpy

  • Importar el tipo de mensaje std_msgs/msg/String ya pre-definido

TipAmbos import representan las dependencias del nodo 
from std_msgs.msg import String

  • Inicializar la librería rclpy

rclpy.init(args=args)

  • Crear un nodo

  rclpy.create_node('<nombre_nodo>')
nodo = rclpy.create_node('publicador')

  • Crear un publicador

  <nodo>.create_publisher( 
      <tipo_mensaje>, '<nombre_topic>', <tamaño_queue>
  )
pub = nodo.create_publisher(String, 'chat', 10)

Declara que el nodo publica mensajes de tipo std_msgs/msg/String sobre un topic llamado chat, y que el tamaño de la cola es 10.

  • Crear un suscriptor

  <nodo>.create_subscription( 
      <tipo_mensaje>, '<nombre_topic>', <funcion_callback>, <tamaño_queue>
  )
sub = nodo.create_subscription(String, 'chat', sub_callback, 10)

  • Crear un timer

  <nodo>.create_timer(<periodo_en_seg>, <funcion_callback>)
timer = nodo.create_timer(1, timer_callback)

  • Continuar ejecución de ROS

  rclpy.spin(<nodo>)
rclpy.spin(nodo)

  • Finalizar la ejecución

rclpy.shutdown()

Actualizar las dependencias

  • Para añadir dependencias a un paquete, modificar el archivo package.xml
package.xml
<package format="3">
    ...
    <exec_depend>rclpy</exec_depend>
    <exec_depend>std_msgs</exec_depend>
    ...
</package>

Declara que el paquete necesita rclpy y std_msgs cuando el código es ejecutado

Instalar las dependencias

rosdep install -i --from-path src --rosdistro jazzy -y
AdvertenciaEjecutar este comando dentro de la carpeta raíz del workspace

Agregar el ejecutable

Para añadir un ejecutable en un paquete de Python, modificar el archivo setup.py

      '<nombre_ejecutable> = <nombre_paquete>.<nombre_archivo>:main',
  ],
},

2.2 Programación de suscriber

El código del suscriber es casi idéntico al del publisher, excepto por la creación de un suscriptor en vez de un publicador, y que no requiere un timer. Recuerda que para que dos nodos se comuniquen, deben coincidir en el mismo topic.

NotaPuede optar por crear un nuevo paquete o un nuevo archivo 
nodo_suscriptor.py
import rclpy

# Importar el tipo de mensaje String
from std_msgs.msg import String

def main(args=None):
    # 1. Inicialización
    # ...

    # 2. Creación de nodo
    # nodo = ...

    # 2.1 Programación de función de callback
    def sub_callback(msg):
        # Mostrar el mensaje en consola
        nodo.get_logger().info('Recibí: "%s"' % msg.data)

    # 2.2 Creación de suscriptor
    # ...

    # 3. Procesamiento de mensajes y callback
    rclpy.spin(nodo)

    # 4. Finalización 
    rclpy.shutdown()


if __name__ == '__main__':
    main()
